Three arcs.
One peak.
Layered ascent to a point.

The mark reads as a structure being built — bottom course, second course, third course, capstone. Each arc emerges in turn, settles, and the peak resolves. Unresolved → ascertained.

Drawn as elliptical-arc segments with squared ends, the outer edges of every layer falling on a single line that converges at the peak. Geometric, not gestural. Scales legibly from 16 px favicon to 14 m signage panel without modification.

// verdict An instrument of ascent, drawn as a tool.
§ 01

Five measurements, one square unit. Stated, not freehand.

240 × 240 canvas
Unit u = 24 (canvas ÷ 10)
All dims in multiples of u
// canonical drawing monochrome · stated
// drawing rules

Five rules. The envelope is the law.

  1. Set the envelope.Draw two lines from the peak tip at (120, 30) to the bottom-arc outer endpoints (17, 218) and (223, 218). Every outer arc endpoint, the triangle base corners, and the peak tip live on these two lines.
  2. Arcs are elliptical segments,not strokes through control points. Each arc is rendered by stroking an SVG elliptical-arc path with stroke-linecap: butt — the tangent at each endpoint is vertical, so the cap reads as a clean horizontal cut.
  3. Bottom arc:rx 92, ry 44, baseline y=218, stroke 22. Middle: rx 60, ry 32, baseline 154, stroke 16. Top: rx 35, ry 18, baseline 104, stroke 12. Each next arc is ~65% of the previous one.
  4. Peakis the only filled element. Sharp isoceles, tip (120, 30), base from (100, 66) to (140, 66). Width 40, height 36.
  5. Stroke ratio is lockedat bottom : middle : top = 22 : 16 : 12 (= 1 : 0.73 : 0.55). When you tune stroke width, the three arcs move as one family — never solo.

Five frames. Foundation, course, course, peak, settle.

Each layer scales up from its own baseline and fades in. The bottom arc anchors first, the courses stack onto it, and the peak resolves last — like a structure being assembled.

Encoded as inline CSS animations inside the SVG itself — no external runtime needed. The file works in any browser, in email, embedded as <img>, dragged into Figma. The motion is the file.

// chapter 01

Foundation.

The first arc is the widest, lowest, most material. It carries the brand even when nothing else is visible. At 16 px favicon scale, the bottom arc is the only thing the eye sees — and that's enough.

Build the foundation right and everything that stacks on it inherits the geometry.

// chapter 02

First course.

The second arc inherits the symmetry axis and reduces by ~30%. It is the same gesture, smaller. The visual rhythm of the mark is the same rhythm of a convergent estimate — successive measurements, each narrower than the last.

Repetition with reduction. The protocol's job, rendered as form.

// chapter 03

Second course.

The third arc is the narrowest of the three. At this point the eye reads the stack as ascent — three layers leaning toward a point that has not yet appeared.

Tension before resolution. The mark earns its peak.

// chapter 04

The peak.

The peak is the only filled, sharp element in the mark. It is the calibrated point — the moment the estimate resolves into a value. Everything below it is process; the peak is the result.

This is also where the brand color does its hardest work. At signage scale, the peak is the first thing visible from across the room.

// chapter 05

Settle.

After the peak resolves, the mark holds its final state for ~3 seconds before the loop restarts. The hold is intentional — it gives the brand presence, not just motion.

Continuously animating logos are noise. A logo that settles is a logo you can read.

§ 07

The numbers. All of them.

copy these into a brand book · paste into figma
// specifications

Mark · color · motion.

// canvas
240 × 240

Square. Aspect 1:1. Padding 20 / 0 / 24 / 0 around the inked silhouette.

// stroke
22 · 16 · 12 · butt

Three locked stroke weights — bottom : middle : top = 22 : 16 : 12. Butt linecaps cut the tangent perpendicular, yielding flat horizontal endpoints. No round-stroke gesture.

// color
#3f8064

oklch(0.55 0.078 156). A kiln-fired sage between forest and verdigris. Single saturation in the system.

// motion
4.2 s loop

1.1 s build · 3.1 s hold. Cubic ease-out, 0.28 s stagger between layers.

// min size
16 px

At 16 px the bottom stroke beefs up to 28 px (in the 240 viewBox) to maintain legibility. Below 16 px, fall back to the filled "peak" glyph alone.

// clear-space
2u · all sides

Equal to bottom arc rise. The mark is never set closer than this to any other element.

// file format
SVG · currentColor

Stroke inherits CSS color, so a single file serves every context. PNG exports at 16 / 32 / 64 / 128 / 256 / 512 / 1024.

// accessibility
role · img

aria-label = "Ascertainty". prefers-reduced-motion respected — animation collapses to static end-state.
